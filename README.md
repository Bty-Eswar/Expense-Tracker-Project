# Expense Tracker Pro

A production-ready full-stack Expense Tracker built using the MERN Stack.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features

- Authentication
- Expense Management
- Dashboard
- Charts
- Budget Tracking
- Search
- Filters
- Responsive UI
- Dark Mode
